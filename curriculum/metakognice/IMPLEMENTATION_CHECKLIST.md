# Implementační Checklist: Sokratovská Mezera

> Krok-za-krokem průvodce pro nasazení scriptu-03 v Activity 4

---

## PŘÍPRAVA - Před Prvním Použitím

### Čtení a Orientace (1-2 dny)

- [ ] Přečíst si **`script-03-socratic-gap.md`** (30 minut)
  - [ ] Pochopit System Prompt
  - [ ] Prostudovat 4 vzorové dialogy
  - [ ] Poznamenat si své otázky

- [ ] Přečíst si **`socratic-facilitator-quick-guide.md`** (20 minut)
  - [ ] Pochopit jednotlivé fáze (PŘED/BĚHEM/PO)
  - [ ] Zapamatovat si pozorovací body
  - [ ] Poznamenat si nejčastější chyby

- [ ] Přečíst si **`SOCRATIC_DIALOG_INDEX.md`** (10 minut)
  - [ ] Pochopit mapování na kurikulum
  - [ ] Projít FAQ
  - [ ] Vědět, kde najít informace

- [ ] Přečíst si **`README_SOCRATIC_UPDATE.md`** (10 minut)
  - [ ] Pochopit, co je nového
  - [ ] Vědět o safeguards

### Technická Příprava

- [ ] Vybrat AI chatbot (doporučeno: Claude, ChatGPT)
- [ ] Mít přístup k internetu během aktivity
- [ ] Připravit počítač/tablet s stabilním připojením
- [ ] Testovat AI s System Promptem (5 minut)
  - [ ] Kopírovat System Prompt ze scriptu
  - [ ] Vložit do AI chatbotu
  - [ ] Testovat s jednou otázkou

### Materiální Příprava

- [ ] Vytisknout **`socratic-student-worksheet.md`**
  - [ ] 1 list na dítě
  - [ ] Preferuje černobílý tisk
  - [ ] Mít k dispozici navíc 1-2 kusy (chyba, druhý pokus)

- [ ] Vytisknout **Pozorovací List**
  - [ ] Z `socratic-facilitator-quick-guide.md`
  - [ ] 1 list na facilitátora
  - [ ] Připravit tužku/pero

- [ ] Vytisknout **Kartičky Typů Otázek** (volitelné)
  - [ ] Z `script-03-socratic-gap.md` Příloha
  - [ ] Lze také mít jako digitální referenci

### Prostředí

- [ ] Vybrat klidné místo bez vyrušování
- [ ] Zajistit, aby byly minimální hlasy na pozadí
- [ ] Přesunout potenciální vyrušující věci pryč
- [ ] Mít k dispozici vodu/zdravý snack

---

## DEN PŘED AKTIVITOU

### Komunikace s Dítětem (15 minut)

- [ ] Informovat dítě o aktivitě
  - [ ] Vysvětlit: "Budeme si hrát Sokratovu hru"
  - [ ] "Já se tě budu ptát, ty si myslíš"
  - [ ] "Nebudu ti dávat odpovědi"
  - [ ] "Cílem je tvé myšlení, ne správné odpovědi"

- [ ] Zeptat se dítěte, zda má otázky
- [ ] Zajistit, aby se dítě cítilo bezpečně
- [ ] Stvrzit čas a místo aktivity

### Příprava Otázky K Zahájení

- [ ] Zvolit jednu iniciální otázku pro dítě
  - [ ] Měla by být otevřená (ne ano/ne)
  - [ ] Měla by se týkat něčeho, co dítě myslí, že je "určitě pravda"
  - [ ] Měla by být na rozmezí kritického myšlení

- [ ] Příklady iniciálních otázek:
  - [ ] "Co si myslíš, jak se nejlépe učit?"
  - [ ] "Jaký typ osoby je dobrý student?"
  - [ ] "Když se myliš, co to znamená?"
  - [ ] "Potřebuješ být chytrý/á, abys uspěl/a?"

### Posledního Check

- [ ] Zkontrolovat spojení internetu
- [ ] Vyzkoušet AI chatbot s System Promptem
- [ ] Mít všechny materiály vyprintované a připravené
- [ ] Seznamit se s pozorovacím listem

---

## DEN AKTIVITY - RÁNO

### 30 Minut Před

- [ ] Připravit místnost (sednout si, vyzkoušet vše)
- [ ] Mít všechny materiály na stole
- [ ] Počítač/tablet připravený (AI chatbot otevřený)
- [ ] Hodinky/alarm na 15 minut (pro časový limit)

### Mentální Příprava Facilitátora

- [ ] Zpamatovat si role (PŘI POZORUJETE, ne pomáhat)
- [ ] Připravit se na ticho (nemusíte se plnit)
- [ ] Připravit se, že dítě nemusí hned odpovídat
- [ ] Věřit v proces (ne v "správné" odpovědi)

---

## BĚHEM AKTIVITY - FÁZE PŘED (5 MINUT)

### Příprava Prostoru

- [ ] Sedět si vedle dítěte
- [ ] Odebrat vyrušující věci
- [ ] Zajistit tichý prostor

### Úvod pro Dítě

- [ ] Řekněte přibližně:
  ```
  "Dneska si zahrajeme zajímavou hru. Já se tě budu ptát,
  a ty si myslíš. Výjimečné je, že já nedám odpovědi.
  Budu se jen ptát otázky, které ti pomohou
  si sám najít odpovědu.

  Tvůj úkol: Myslej pečlivě. Nejde o správné odpovědi,
  jde o to, aby si byl/a vědom/á, jak myslíš."
  ```

- [ ] Zeptat se: "Máš otázky?"
- [ ] Stvrzovat odpovědi (ne posuzovat)

### Iniciální Otázka

- [ ] Položit zvolenu otázku k zahájení
- [ ] Čekat na odpověď (min. 5 sekund)
- [ ] Zapsat si odpověď na pozorovací list
- [ ] Říci: "Skvělé! Pojďme se na to podívat s AI"

---

## BĚHEM AKTIVITY - FÁZE BĚHEM (15 MINUT)

### Start AI Dialogu

- [ ] Zkopírovat System Prompt do AI chatbotu
- [ ] Vložit iniciální tvrzení dítěte
- [ ] Spustit AI ("Send" nebo Enter)

### Vaše Role - POUZE Pozorování

- [ ] Seďte tichě
- [ ] Sledujte výraz dítěte
  - [ ] Je zvědavé?
  - [ ] Je zmatené?
  - [ ] Je frustráciované?
  - [ ] Je soustředěné?

- [ ] Zapisujte do pozorovacího listu
  - [ ] Které otázky AI pokládá
  - [ ] Jak dítě odpovídá
  - [ ] Vaše odbornosti
  - [ ] Reakte dítěte

- [ ] Čekejte na přirozené konce (dítě přestane psát)
- [ ] Pokud dítě přestane, nechte ho psát další odpověď

### Intervence - POUZE Pokud Nutné

#### Pokud Dítě Neví Odpověď
- [ ] Čekej déle (30 sekund)
- [ ] Řekni: "Bez spěchu, přemýšlej si"
- [ ] Nech AI položit jednodušší otázku

#### Pokud AI Odchází z Tématu
- [ ] Tichě to zaznameň
- [ ] Napiš: "Pojďme se vrátit na téma..."
- [ ] Přesměruj dialog

#### Pokud Dítě Se Frustráciuje (>2 min)
- [ ] Řekni: "Vidím, že je to těžké. Pauzujeme?"
- [ ] PAUSE dialog
- [ ] Nabídni Script 1 (Teachable AI) nebo bez AI
- [ ] Zaznamenej v pozorovacím listu

### Časový Management

- [ ] Spusť alarm na 15 minut
- [ ] V 10. minutě (pokud stojí): "Zbývá 5 minut"
- [ ] V 15. minutě: STOP dialog
  - [ ] Řekni: "Skvělá práce! Teď si odpočineme"

---

## BĚHEM AKTIVITY - FÁZE PO (10 MINUT)

### Reflexní Otázky

- [ ] Zeptat se dítěte:
  ```
  1. "Co sis právě uvědomil/a o sobě?"
  2. "Které otázky AI ti více pomohly?"
  3. "Změnil ses během dialogu?"
  4. "Co bys udělal/a příště?"
  ```

- [ ] Naslouchat odpovědím
- [ ] Neposkytovat své názory
- [ ] Stvrzovat jejich myšlení ("To je zajímavá pozorování")

### Pracovní List

- [ ] Dát dítěti **`socratic-student-worksheet.md`**
- [ ] Řekni: "Teď si zapiš, co sis myslel/a"
- [ ] Nech si dítě čas na vyplnění (5-10 minut)
- [ ] Být k dispozici pro otázky

### Finální Poznámka

- [ ] Řekni: "Jsem na tebe pyšný/á za tvou práci"
- [ ] "Vidíš, jak sis myslel/a hlouběji?"
- [ ] Stvrzit jejich úsilí (ne správnost)

---

## PO AKTIVITĚ

### Okamžitě (v ten den)

- [ ] Sbrat worksheet od dítěte
- [ ] Přečíst si, co dítě napsalo
- [ ] Zaznamenat klíčové pozorování
- [ ] Zapsat si poznámky do pozorovacího listu (co se dařilo, co ne)

### Následující Den

- [ ] Přečíst si všechny poznámky
- [ ] Identifikovat, co se dařilo
  - [ ] Která otázka AI byla nejúčinnější?
  - [ ] V kterém bodě se dítě zlomilo?
  - [ ] Vidíš změnu myšlení?

- [ ] Identifikovat, co se nedařilo
  - [ ] Byla některá otázka moc těžká?
  - [ ] Frustrácioval se v určitém bodě?
  - [ ] Byl čas příliš dlouhý/krátký?

- [ ] Plánovat příští dialog (Activity 4 opakování)

### Dlouhodobě

- [ ] Sledovat, jak dítě používá Sokratovský dialog
  - [ ] Klád si sama otázky?
  - [ ] Aplikuje to v jiných oblastech?
  - [ ] Stala se metakognice automatickou?

- [ ] Zaznamenávat pokrok v deníku
  - [ ] Porovnávat Activity 1 → Activity 4
  - [ ] Vidíš rozvoj kritického myšlení?

---

## CHECKLIST PRO PRVNÍ POKUS

### Před Aktivitou
- [ ] Přečteno všech 5 dokumentů
- [ ] AI chatbot vyzkoušen
- [ ] Materiály vytisknuty (2 kusy worksheetu)
- [ ] Otázka k zahájení vybraná
- [ ] Dítě informováno a ochotné

### Během Aktivity
- [ ] PŘED (5 min): Úvod + iniciální otázka
- [ ] BĚHEM (15 min): Dialog s AI (vy pozorujete)
- [ ] PO (10 min): Reflexe + worksheet

### Po Aktivitě
- [ ] Worksheet sbran a přečten
- [ ] Poznámky zaznamenány
- [ ] Feedback od dítěte sbran (ústně)

### Měřítko Úspěchu
- [ ] Dítě pochopilo, že AI se ptá (ne odpovídá)
- [ ] Dítě odpovídalo otázkám (bylo zapojené)
- [ ] Dítě si psalo něco do worksheetu
- [ ] Dítě nebylo frustráciované (nebo jen chvílkově)
- [ ] Facilitátor zůstal v pozorování (ne pomáhání)

---

## BEZPEČNOSTNÍ CHECKLIST

### Před každou Aktivitou
- [ ] Dítě je odpočaté (ne vyčerpané)
- [ ] Dítě má vodu/snack
- [ ] Místo je tichý a bezpečný
- [ ] Facilitátor je klidu (ne unaven/a)

### Během Aktivity
- [ ] Facilitátor pozoruje znaky stresu/frustrace
- [ ] Alarm je nastavený (15 minut)
- [ ] AI zůstává v tématu metakognice
- [ ] Dítě se cítí bezpečně (neposuzované)

### Po Aktivitě
- [ ] Dítě není nevyrovnané (pozitivní nálada)
- [ ] Dítě má dostatek času na reflexi
- [ ] Facilitátor si zaznamenává poznámky
- [ ] Plán pro příští dialog

---

## TROUBLESHOOTING

### Pokud Dítě Neví Odpověď
- [ ] Čekej (30 sekund bez řeči)
- [ ] Řekni: "Bez spěchu"
- [ ] Nech AI položit jednodušší otázku
- [ ] Pokud stále neví: "Je to těžké, OK"

### Pokud Dítě Se Frustráciuje
- [ ] Přestanu seanci
- [ ] Nabídni přestávku
- [ ] Vrať se k méně Sokratovskému přístupu
- [ ] Zaznamenej v pozorovacím listu

### Pokud AI Pokládá Stejnou Otázku 3x
- [ ] Zaznamenej to
- [ ] Měl/a bys změnit typ otázky (v průvodci)
- [ ] Pokud AI neodpovídá, resetuj dialog

### Pokud Máme Méně Času (15 minut)
- [ ] Zkrátit PŘED (2 min)
- [ ] BĚHEM (10 min)
- [ ] PO (3 min)
- [ ] Worksheet na doma (volitelně)

### Pokud Máme Více Času (45 minut)
- [ ] PŘED (5 min)
- [ ] BĚHEM (20 min)
- [ ] Přestávka (5 min)
- [ ] PO (15 min)
- [ ] Reflektivní deník (10 min)
