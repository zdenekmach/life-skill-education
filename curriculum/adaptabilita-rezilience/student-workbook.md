## Lekce 1: Síla JEŠTĚ

### Dva typy myšlení

```
╔═══════════════════════════════════════════════════════════════╗
║                                                               ║
║     🔒 FIXNÍ              vs.        🌱 RŮSTOVÉ              ║
║                                                               ║
║  "Jsem prostě takový"         "Můžu se zlepšit"              ║
║  "Neumím to"                  "JEŠTĚ to neumím"              ║
║  "Chyba = jsem hloupý"        "Chyba = příležitost"          ║
║  "Talent je vše"              "Úsilí vede k růstu"           ║
║                                                               ║
╚═══════════════════════════════════════════════════════════════╝
```

### Síla slova JEŠTĚ

```
"Neumím to" → "JEŠTĚ to neumím"

Jedno slovo mění všechno!
```

## Lekce 3: Surfař změn

### 5 fází změny

```
1. ŠOK         "Co?! To ne!"
2. POPŘENÍ     "Možná se to změní zpátky"
3. FRUSTRACE   "Proč zrovna já?"
4. EXPERIMENTOVÁNÍ "OK, zkusím to nějak"
5. PŘIJETÍ     "Zvládl/a jsem to, jdu dál"

Všechny fáze jsou NORMÁLNÍ!
```

### ADAPT Framework

```
A = ACCEPT      Přijmi realitu
D = DETERMINE   Urči, co můžeš ovlivnit
A = ADJUST      Přizpůsob plány
P = PERSIST     Vytrvej
T = TRANSFORM   Najdi příležitost
```
