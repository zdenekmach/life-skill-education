# Modul: Adaptabilita a rezilience

## Přehled

Modul Adaptabilita a rezilience učí děti zvládat změny, čelit překážkám a rozvíjet psychickou odolnost. V rychle se měnícím světě jsou tyto dovednosti klíčové pro úspěch i osobní pohodu.

## Cíle modulu

Po absolvování modulu žáci:
1. Rozumí konceptu růstového myšlení
2. Dokáží konstruktivně přijímat neúspěch
3. Umí se adaptovat na změny
4. Znají strategie pro zvládání stresu
5. Rozvíjí kreativitu a inovativní myšlení

## Klíčové koncepty

### Growth Mindset (Růstové myšlení)

```
╔═══════════════════════════════════════════════════════════════╗
║                    RŮSTOVÉ MYŠLENÍ                            ║
╠═══════════════════════════════════════════════════════════════╣
║                                                               ║
║   FIXED (fixní)           vs.    GROWTH (růstové)            ║
║   ─────────────────────────────────────────────────          ║
║   "Jsem prostě takový"           "Můžu se zlepšit"           ║
║   "Nedokážu to"                  "Ještě to nedokážu"         ║
║   "Chyba = jsem hloupý"          "Chyba = příležitost"       ║
║   "Talent je vše"                "Úsilí vede k růstu"        ║
║                                                               ║
╚═══════════════════════════════════════════════════════════════╝
```

### BOUNCE Framework (Rezilience)

```
B = BREATHE     Zastav se a dýchej
O = OBSERVE     Pozoruj, co se děje
U = UNDERSTAND  Pochop situaci
N = NAVIGATE    Najdi cestu vpřed
C = CONNECT     Spoj se s ostatními
E = EVOLVE      Uč se a rostu
```
