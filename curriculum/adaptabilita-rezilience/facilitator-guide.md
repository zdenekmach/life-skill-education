# Průvodce facilitátora: Adaptabilita a rezilience

## O tomto modulu

Modul Adaptabilita a rezilience učí děti zvládat změny, čelit překážkám a rozvíjet psychickou odolnost. Jde o jeden z nejdůležitějších modulů pro osobní rozvoj.

## Aktivita 1: Síla JEŠTĚ

### Cíl
Naučit koncept růstového myšlení a techniku transformace pomocí "JEŠTĚ".

### Klíčové koncepty

**Fixed vs. Growth Mindset (Carol Dweck):**
- Fixed: "Schopnosti jsou dané"
- Growth: "Schopnosti se rozvíjejí úsilím"

**Síla slova JEŠTĚ:**
- "Neumím to" → "JEŠTĚ to neumím"
- Mění definitvní tvrzení na proces

### Průběh

1. **Úvod:** Neuroplasticita - mozek se mění celý život
2. **Teorie:** Dva typy myšlení, příklady
3. **Praxe:** Třídění výroků, transformace pomocí JEŠTĚ
4. **Reflexe:** Osobní růstové závazky

### Tipy

- Nikdo nemá 100% růstové myšlení
- Oceňuj snahu, ne jen výsledek
- Příběhy slavných "neúspěchů" motivují

## Aktivita 2: Příběh neúspěchu

### Cíl
Naučit vnímat neúspěch jako součást učení, ne jako konec.

### Příprava

```
⚠️ DŮLEŽITÉ:
Facilitátor sdílí VLASTNÍ příběh neúspěchu PRVNÍ!
To vytváří bezpečné prostředí pro ostatní.
```

### Průběh

1. **Úvod:** "Fail Wall of Fame" - slavní a jejich selhání
2. **Teorie:** Neúspěch jako data, ne verdikt
3. **Praxe:** Sdílení příběhů (dobrovolné!), analýza lekcí
4. **Reflexe:** Technika Reframe, třídní dohoda

### Citlivé situace

- Někteří žáci mohou mít těžké zkušenosti
- Nenutit ke sdílení
- Respektovat "pass"
- Mít připravený individuální rozhovor

### Klíčové sdělení

```
"Selhal jsem v úkolu" ≠ "Jsem selhání"
```

## Aktivita 3: Surfař změn

### Cíl
Naučit adaptovat se na změny pomocí křivky změny a ADAPT frameworku.

### Křivka změny (5 fází)

```
1. Šok      - "Co?!"
2. Popření  - "To ne!"
3. Frustrace - "Proč já?"
4. Experimentování - "OK, zkusím"
5. Přijetí  - "Jdu dál"
```

### ADAPT Framework

```
A = Accept (Přijmi realitu)
D = Determine (Urči, co můžeš ovlivnit)
A = Adjust (Přizpůsob plány)
P = Persist (Vytrvej - křivka jde nahoru)
T = Transform (Najdi příležitost)
```

### Kruhy vlivu

Učíme děti rozlišovat:
- Co MŮŽU ovlivnit (reakce, úsilí, přístup)
- Co NEMŮŽU ovlivnit (rozhodnutí druhých, počasí, minulost)

→ Zaměř energii na ovlivnitelné!

## Aktivita 4: Moje odolnost

### Cíl
Naučit strategie pro budování psychické odolnosti.

### BOUNCE Framework

```
B = Breathe    (Dýchej - technika 4-7-8)
O = Observe    (Pozoruj - co se děje, co cítím)
U = Understand (Pochop - proč se to stalo)
N = Navigate   (Naviguj - jaké mám možnosti)
C = Connect    (Spoj se - kdo mi může pomoct)
E = Evolve     (Rozvíjej se - co se naučím)
```

### Dechová technika 4-7-8

```
Nádech:   4 sekundy (nosem)
Zadržení: 7 sekund
Výdech:   8 sekund (ústy)
Opakovat: 3-4×
```

### Důležité upozornění

```

• Vážné šikaně
• Zneužití
```

## Aktivita 5: Inovační výzva

### Cíl
Aplikovat růstové myšlení a kreativitu na reálný problém.

### Design Thinking proces

```
1. EMPATIE    - Pochop uživatele
2. DEFINICE   - Jasně definuj problém
3. IDEACE     - Brainstorm nápadů
4. PROTOTYP   - Vytvoř rychlou verzi
5. TEST       - Získej zpětnou vazbu
6. ITERACE    - Vylepši na základě dat
```

### Hodnocení

Důraz na PROCES, ne jen výsledek:
- Jak překonal/a překážky?
- Jak reagoval/a na zpětnou vazbu?
- Co se naučil/a?

## FAQ

**Q: Co když dítě odmítá sdílet neúspěch?**
A: Respektuj. Nabídni alternativu (napsat si pro sebe, nebo sdílet obecný příklad).

**Q: Jak řešit perfekcionismus?**
A: Oceňuj proces a úsilí. Zdůrazni, že chyby jsou normální. Vlastní příklad.

**Q: Je BOUNCE dostatečný pro vážné problémy?**
A: Ne. BOUNCE je pro běžné výzvy. Vážné problémy vyžadují odbornou pomoc.

**Q: Jak zapojit rodiče?**
A: Sdílej materiály. Navrhni rodinné aktivity (sdílení neúspěchů u večeře).
