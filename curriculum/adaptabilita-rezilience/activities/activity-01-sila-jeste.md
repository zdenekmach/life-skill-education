# Aktivita 1: Síla JEŠTĚ

## Přehled

### Cíl
Naučit koncept růstového myšlení (growth mindset) a sílu slova "JEŠTĚ".

### Výstupy
Po aktivitě žáci:
- Rozumí rozdílu mezi fixním a růstovým myšlením
- Umí transformovat negativní tvrzení pomocí "JEŠTĚ"
- Chápou, že schopnosti lze rozvíjet úsilím
- Identifikují vlastní fixní myšlenky

### Časový plán
| Fáze | Čas | Aktivita |
|------|-----|----------|
| Úvod | 5 min | Hádanka s mozkem |
| Teorie | 10 min | Dva typy myšlení |
| Praxe | 20 min | Transformace tvrzení |
| Reflexe | 10 min | Moje růstové závazky |

---

## Průběh aktivity

### 1. Úvod: Hádanka s mozkem (5 min)

**Otázka pro třídu:**

```
Co má víc spojů než hvězd ve vesmíru?
.
.
.
Odpověď: TVŮJ MOZEK!

Tvůj mozek má asi 86 miliard neuronů.
A každý se může spojit s tisíci dalšími!
```

**Klíčové sdělení:**
- Mozek se mění celý život (neuroplasticita)
- Učením se vytváří nová spojení
- To, co "neumíš", můžeš naučit

### 2. Teorie: Dva typy myšlení (10 min)

#### Carol Dweck a růstové myšlení

```
╔═══════════════════════════════════════════════════════════════╗

║   PŘI CHYBĚ:               PŘI CHYBĚ:                        ║

║                                                               ║
║   PŘI VÝZVĚ:               PŘI VÝZVĚ:                        ║
║   "Je to moc těžké"        "Je to příležitost růst"          ║
```

### 3. Praxe: Transformace tvrzení (20 min)

#### Cvičení 1: Rozpoznej typ myšlení (5 min)

**Rozdej kartičky s tvrzeními. Žáci třídí na FIXED a GROWTH:**

| Tvrzení | Fixed | Growth |
|---------|-------|--------|
| "Matematika mi prostě nejde" | ✓ | |
| "Když udělám chybu, zkusím jinak" | | ✓ |
| "Někdo to buď má, nebo nemá" | ✓ | |
| "Těžké úkoly mě posouvají" | | ✓ |
| "Nechci vypadat hloupě" | ✓ | |
| "Zpětná vazba mi pomáhá růst" | | ✓ |

#### Cvičení 2: Transformace pomocí JEŠTĚ (10 min)

**Pracovní list:**

```
TRANSFORMUJ POMOCÍ "JEŠTĚ"

FIXED tvrzení              →    GROWTH tvrzení

"Neumím anglicky"          →    "________________________"

"Nejsem sportovec"         →    "________________________"

"Nemám kreativní nápady"   →    "________________________"

"Nedokážu mluvit před lidmi" →  "________________________"

"Programování je moc těžké" →   "________________________"
```

**Vzorové odpovědi:**
- "JEŠTĚ neumím anglicky tak, jak bych chtěl/a"
- "JEŠTĚ nejsem sportovec, ale můžu trénovat"

#### Cvičení 3: Moje fixní myšlenky (5 min)

```
MOJE FIXNÍ MYŠLENKY

Napiš 3 věci, o kterých si myslíš "to neumím" nebo "to nejde":

1. ________________________________________________

2. ________________________________________________

3. ________________________________________________

Teď přidej JEŠTĚ:

1. JEŠTĚ ___________________________________________

2. JEŠTĚ ___________________________________________

3. JEŠTĚ ___________________________________________
```

### 4. Reflexe: Moje růstové závazky (10 min)

#### Diskuse

```
DISKUSNÍ OTÁZKY:

1. Kdy jsi naposledy něco nevzdal/a i když to bylo těžké?

2. Co se stane, když si řekneš "JEŠTĚ neumím" místo "neumím"?

3. Kdo je někdo, koho obdivuješ? Myslíš, že jim to šlo
   vždycky snadno, nebo tvrdě pracovali?
```

## Poznámky pro facilitátora

### Důležité body
- Nikdo nemá 100% růstové myšlení - všichni máme fixní oblasti
- Cílem není kritizovat, ale uvědomit si a transformovat
- Oceň snahu, ne jen výsledek

### Časté mylné představy
| Mýtus | Realita |
|-------|---------|
| Růstové myšlení = pozitivní myšlení | Ne, jde o přístup k učení a chybám |
| Talent neexistuje | Existuje, ale není limitující |
| Stačí říct "JEŠTĚ" | Slovo je začátek, ne řešení |

### Propojení s domovem
- Navrhni, aby rodiny používali "JEŠTĚ" doma
- Sdílej příběhy slavných "neúspěchů"
