# Aktivita 2: Můj příběh neúspěchu

## Přehled

### Cíl
Naučit děti vnímat neúspěch jako součást učení a růstu, ne jako konec.

### Výstupy
Po aktivitě žáci:
- Chápou, že neúspěch je normální součást života
- Umí analyzovat, co se z neúspěchu naučili
- Dokáží sdílet zkušenosti s neúspěchem bez studu
- Vidí neúspěch jako zpětnou vazbu, ne jako verdikt

### Časový plán
| Fáze | Čas | Aktivita |
|------|-----|----------|
| Úvod | 5 min | "Fail Wall of Fame" |
| Teorie | 10 min | Neúspěch jako učitel |
| Praxe | 20 min | Moje selhání, moje lekce |
| Reflexe | 10 min | Reframe - nový pohled |

### 2. Teorie: Neúspěch jako učitel (10 min)

#### Dva pohledy na neúspěch

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│     POHLED A: Neúspěch = konec                             │
│     ─────────────────────────                               │
│     "Selhal jsem" = "Jsem selhání"                         │
│     Důsledek: Vzdám to, už to nezkusím                     │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│     POHLED B: Neúspěch = data                              │
│     ─────────────────────────                               │
│     "Selhal jsem" = "Zjistil jsem, co nefunguje"          │
│     Důsledek: Zkusím to jinak                              │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### FAIL = First Attempt In Learning

```
F = First      (První)
A = Attempt    (Pokus)
I = In         (V)
L = Learning   (Učení)

FAIL = První pokus v učení
```

#### Cyklus učení z neúspěchu

```
        ┌─────────┐
        │  POKUS  │
        └────┬────┘
             │
             ▼
        ┌─────────┐
        │ SELHÁNÍ │
        └────┬────┘
             │
             ▼
     ┌───────────────┐
     │   REFLEXE     │
     │ Co se stalo?  │
     │ Co jsem se    │
     │ naučil/a?     │
     └───────┬───────┘
             │
             ▼
     ┌───────────────┐
     │   ÚPRAVA      │
     │ Co udělám     │
     │ jinak?        │
     └───────┬───────┘
             │
             ▼
    ┌────────────────┐
    │  NOVÝ POKUS    │
    └────────────────┘
```

### 3. Praxe: Moje selhání, moje lekce (20 min)

#### Cvičení 1: Příběhy selhání - bezpečné sdílení (10 min)

**Facilitátor první sdílí vlastní příběh selhání!**

**Struktura sdílení:**
```
MŮJ PŘÍBĚH SELHÁNÍ

1. CO SE STALO:
   _________________________________________________

2. JAK JSEM SE CÍTIL/A:
   _________________________________________________

3. CO JSEM SE NAUČIL/A:
   _________________________________________________

4. CO JSEM UDĚLAL/A JINAK:
   _________________________________________________

5. JAK TO DOPADLO NAKONEC:
   _________________________________________________
```

**Pravidla sdílení:**
- Sdílíme dobrovolně
- Nekomentujeme, neodsuzujeme
- Respektujeme odvahu sdílet
- Můžeme říct "pass"

### 4. Reflexe: Reframe - nový pohled (10 min)

#### Technika Reframe

```
REFRAME = Dát situaci nový rámec

STARÉ VNÍMÁNÍ              →    NOVÉ VNÍMÁNÍ
─────────────────────────────────────────────
"Dostal jsem špatnou       →    "Vím, co zlepšit
známku z testu"                  příště"

"Nevzali mě do týmu"       →    "Můžu trénovat
                                 a zkusit znovu"

"Pokazil jsem prezentaci"  →    "Příště se líp
                                 připravím"
```

#### Cvičení: Reframe moje selhání

```
MŮJ REFRAME

Moje selhání:
_________________________________________________

Staré vnímání (jak jsem to viděl/a):
_________________________________________________

Nové vnímání (co dobrého z toho vzejde):
_________________________________________________
```

## Poznámky pro facilitátora

### Důležité body
- VŽDY sdílej vlastní příběh selhání první
- Vytvoř bezpečné prostředí bez posuzování
- Respektuj, když někdo nechce sdílet

### Citlivé situace

Někteří žáci mohou mít:
- Traumatické zkušenosti se selháním
- Perfekcionismus (strach z jakékoliv chyby)
- Rodinné prostředí netolerantní k chybám

**Jak reagovat:**
- Nenutit ke sdílení
- Nabídnout individuální rozhovor
- Odkázat na školního psychologa, pokud potřeba

### Důležité rozlišení

```
SELHÁNÍ ≠ OSOBNÍ SELHÁNÍ

Selhal jsem v úkolu ≠ Jsem selhání
Udělal jsem chybu ≠ Jsem chyba
Neuspěl jsem tentokrát ≠ Nikdy neuspěju
```
