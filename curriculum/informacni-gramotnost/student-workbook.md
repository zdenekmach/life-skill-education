## Lekce 1: Hledač pokladů

### Vyhledávací triky

**Trik 1: Uvozovky = přesná fráze**
```
"změna klimatu" najde přesně tato dvě slova vedle sebe
změna klimatu najde stránky obsahující obě slova kdekoliv
```

**Trik 2: Minus = vyloučení**
```
jaguár -auto najde zvíře, ne značku auta
apple -iphone najde ovoce, ne telefon
```

**Trik 3: site: = konkrétní web**
```
site:wikipedia.org Praha najde pouze na Wikipedii
site:gov.cz dotace najde pouze na vládních webech
```

**Trik 4: filetype: = typ souboru**
```
filetype:pdf fyzika najde PDF dokumenty o fyzice
```

## Lekce 3: Fake nebo fakt?

### ESCAPE Framework

Když něco vypadá podezřele, použij ESCAPE:

```
┌─────────────────────────────────────────────────────────┐
│                    🔎 ESCAPE 🔎                         │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   E = EVIDENCE (Důkazy)                                │
│       Jsou tam nějaké důkazy?                          │
│       Odkud pocházejí?                                  │
│                                                         │
│   S = SOURCE (Zdroj)                                   │
│       Kdo to sdílí?                                    │
│       Je to důvěryhodný zdroj?                         │
│                                                         │
│   C = CONTEXT (Kontext)                                │
│       Co chybí?                                         │
│       Je to vytržené z kontextu?                       │
│                                                         │
│   A = AUDIENCE (Publikum)                              │
│       Na koho to cílí?                                 │
│       Koho to má přesvědčit?                           │
│                                                         │
│   P = PURPOSE (Účel)                                   │
│       Proč to někdo sdílí?                             │
│       Informovat? Prodat? Vystrašit?                   │
│                                                         │
│   E = EXECUTION (Provedení)                            │
│       Jak to vypadá?                                   │
│       Profesionální nebo podezřelé?                    │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

---

### Varovné znaky dezinformací

```
🚩 POZOR, KDYŽ VIDÍŠ:

□ ŠOKUJÍCÍ TITULEK - "Tohle vás zničí!" "Vědci v šoku!"
□ CHYBÍ AUTOR - Nikdo se k tomu nehlásí
□ CHYBÍ DATUM - Nevíš, kdy to bylo napsáno
□ EMOCIONÁLNÍ JAZYK - Snaží se tě naštvat nebo vystrašit
□ ŽÁDNÉ ZDROJE - "Vědci říkají..." Jací vědci?
□ PŘÍLIŠ DOBRÉ, ABY TO BYLA PRAVDA - "Zhubnete 10 kg za týden!"
□ VYBÍZÍ K RYCHLÉMU SDÍLENÍ - "Sdílej, než to smažou!"
```
