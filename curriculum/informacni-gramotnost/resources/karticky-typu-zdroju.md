# Kartičky: Typy zdrojů informací

## Jak používat

- Vytiskni, rozstříhej a (volitelně) zalaminuj
- Použij pro cvičení rozpoznávání typů zdrojů
- Hrajte "Roztřiď zdroje"
- Diskutujte o výhodách a nevýhodách každého typu

---

## Kategorie 1: Primární vs. sekundární zdroje

### Kartička 1: PRIMÁRNÍ ZDROJ

```
┌─────────────────────────────────────┐
│                                     │
│         📜 PRIMÁRNÍ ZDROJ           │
│                                     │

│                                     │
├─────────────────────────────────────┤
│                                     │
│  PŘÍKLADY:                          │
│  • Vědecká studie (originální)     │
│  • Rozhovor s očitým svědkem       │

│  • Fotografie z události           │

│                                     │
│  VÝHODA: Nezkresklené, původní     │
│  NEVÝHODA: Může být těžké najít    │
│                                     │

```

## Hra: Roztřiď zdroje

**Pravidla:**
1. Zamíchej kartičky
2. Táhni jednu a přečti příklad
3. Ostatní hádají typ zdroje
4. Diskutujte důvěryhodnost

**Příklady k třídění:**

