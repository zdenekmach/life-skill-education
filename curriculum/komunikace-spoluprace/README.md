# Modul: Komunikace a spolupráce

## Přehled

Tento modul učí děti **efektivně komunikovat** a **produktivně spolupracovat** s ostatními. Zaměřuje se na aktivní naslouchání, asertivní komunikaci, řešení konfliktů a týmovou spolupráci.

## Cíle modulu

Po dokončení modulu dítě:

1. **Aktivně naslouchá** - plně se soustředí na mluvčího, parafrázuje a klade otázky
2. **Komunikuje asertivně** - vyjadřuje své potřeby a názory bez agrese i pasivity
3. **Řeší konflikty konstruktivně** - používá metodu WIN-WIN pro nalezení oboustranně přijatelných řešení
4. **Spolupracuje v týmu** - přebírá různé role, přispívá k cíli, respektuje ostatní
5. **Dává a přijímá zpětnou vazbu** - konstruktivně a s respektem

## Klíčové koncepty

### 1. Aktivní naslouchání

```
SLYŠET ≠ NASLOUCHAT

Slyšet = fyzicky vnímat zvuky
Naslouchat = plně se soustředit + pochopit + reagovat

Techniky:
- Oční kontakt
- Přitakávání
- Parafrázování ("Takže říkáš, že...")
- Otázky ("Co tím myslíš?")
```

### 2. Komunikační styly

```
PASIVNÍ          ASERTIVNÍ         AGRESIVNÍ
   ↓                 ↓                 ↓
"Já nejsem       "Já jsem OK,      "Já jsem OK,
 důležitý"       ty jsi OK"        ty nejsi OK"

Ustupuji         Respektuji         Útočím
Mlčím            Vyjadřuji se       Křičím
Potlačuji        Vyjednávám         Vnucuji
```

### 3. JÁ-výroky

```
╔═══════════════════════════════════════════════════════════╗
║                     STRUKTURA JÁ-VÝROKU                   ║
╠═══════════════════════════════════════════════════════════╣
║                                                           ║
║  "Když [popis situace],                                  ║
║   cítím se [emoce],                                      ║
║   protože [důvod].                                       ║
║   Přál/a bych si [žádost]."                             ║
║                                                           ║
╚═══════════════════════════════════════════════════════════╝

Příklad:
TY-výrok: "Ty mě nikdy neposloucháš!"
JÁ-výrok: "Když se díváš do mobilu, když mluvím,
          cítím se ignorovaný/á,
          protože mám pocit, že tě nezajímám.
          Přál/a bych si, abys odložil/a mobil."
```

### 4. Řešení konfliktů (WIN-WIN)

```
Krok 1: STOP - Zastavit se, uklidnit se
Krok 2: POSLOUCHAT - Pochopit druhou stranu
Krok 3: VYJÁDŘIT - Říct svůj pohled (JÁ-výrok)
Krok 4: HLEDAT - Najít řešení, kde oba vyhrávají
Krok 5: DOMLUVIT - Shodnout se na konkrétním řešení
```

### 5. Týmové role

```
ROLE V TÝMU (zjednodušeně):

👑 LÍDR - organizuje, motivuje
💡 KREATIVEC - přináší nápady
📋 REALIZÁTOR - dotahuje do konce
🔍 ANALYTIK - kontroluje kvalitu
🤝 HARMONIZÁTOR - udržuje dobré vztahy

Každý tým potřebuje MIX rolí!
```

### 6. Zpětná vazba (Feedback)

```
SENDVIČOVÁ METODA:

🍞 Pozitivum: Co se povedlo
🥩 Konstruktivní: Co zlepšit + JAK
🍞 Pozitivum: Povzbuzení

Pravidla:
- Konkrétní (ne "bylo to dobré")
- O chování, ne o osobě
- S návrhem zlepšení
```

## Očekávané výstupy

### Měřitelné dovednosti

1. **Parafrázování** - Dítě dokáže přeformulovat sdělení druhého
2. **JÁ-výroky** - Dítě používá strukturu "Když... cítím se... protože... přál/a bych si..."
3. **WIN-WIN** - Dítě navrhne řešení přijatelné pro obě strany
4. **Týmová role** - Dítě reflektuje svůj přínos týmu
5. **Konstruktivní feedback** - Dítě formuluje specifickou zpětnou vazbu s návrhem

### Certifikace

Po úspěšném dokončení modulu dítě obdrží:
- **Certifikát Komunikace a spolupráce**
- **Badge: Týmový hráč** (volitelně)

## Poznámky pro implementaci

### Pro koho je modul vhodný

- Děti 11-13 let (5.-8. třída)
- Funguje individuálně i ve skupině
- Skupinové aktivity preferovány pro Aktivitu 4

### Citlivá témata

- Konflikty v rodině - neptat se na detaily
- Šikana - přesměrovat na odborníka
- Komunikační poruchy - adaptovat aktivity

### Úpravy pro věk

- 11-12 let: Více příkladů, jednodušší konflikty
- 13+ let: Komplexnější situace, hlubší reflexe
