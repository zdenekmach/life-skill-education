## Lekce 1: Strážce hesla

### Co dělá heslo silným?

```
✅ SILNÉ HESLO:
□ 12+ znaků (čím víc, tím líp)
□ Mix velkých a malých písmen
□ Čísla
□ Symboly (!@#$%...)
□ Není to běžné slovo
□ Jiné pro každý účet!
```

### Metoda 1: Passphrase (heslo-věta)

```
1. Vymysli větu: "Můj pes Rek má 4 nohy a miluje piškoty"
2. Vezmi první písmena: MpRm4namP
3. Přidej symboly: MpRm4namP!@

VÝSLEDEK: 12 znaků, silné, zapamatovatelné!
```

### Metoda 2: 4 náhodná slova

```
Koláč-Strom-Modrý-7

= 18 znaků, snadno zapamatovatelné, těžké prolomit
```

## Lekce 2: Digitální stopa

### Co je digitální stopa?

```
AKTIVNÍ STOPA (vědomě vytváříš):
• Příspěvky, fotky, komentáře
• Profily, které vyplníš

PASIVNÍ STOPA (vzniká automaticky):
• Cookies, historie, lokace
```

### Pravidlo babičky a zaměstnavatele

Před sdílením se zeptej:
1. "Chtěl/a bych, aby tohle viděla moje babička?"
2. "Chtěl/a bych, aby tohle viděl můj budoucí zaměstnavatel?"

Pokud NE → NESDÍLEJ

## Lekce 3: AI gramotnost

### Jak AI funguje?

```
1. DATA (obrovské množství textu)
       ↓
2. TRÉNINK (hledání vzorů)
       ↓
3. MODEL (naučené vzory)
       ↓
4. VÝSTUP (predikce "co by mělo následovat")
```

### Co AI umí a neumí

```
✅ UMÍ:                    ❌ NEUMÍ:
• Zpracovat data rychle   • Skutečně rozumět
• Najít vzory             • Mít pocity nebo názory
• Generovat text          • Být vždy přesná
• Překládat               • Znát aktuální události
• Vysvětlovat             • Nést zodpovědnost
```

## Lekce 4: Rozpoznávání podvodů

### SCAM Framework

```
S = SUSPICIOUS SENDER    Podezřelý odesílatel
C = CREATE URGENCY       Vytvářejí tlak
A = ASK FOR INFO         Žádají informace
M = MISMATCH             Něco nesedí
```

---

### Cvičení 4.1: Analyzuj email

```
Od: security@paypa1.com
Předmět: ⚠️ VAROVÁNÍ: Váš účet bude zablokován!

Vážený zákazníku,
Zaznamenali jsme podezřelou aktivitu. IHNED ověřte
svou identitu kliknutím na odkaz níže, jinak bude
váš účet trvale zablokován do 24 hodin!

[OVĚŘIT ÚČET]
```

**SCAM analýza:**
```
S - Podezřelý odesílatel?  □ Ano □ Ne
    Co je špatně? _______________________________

C - Vytváří tlak?          □ Ano □ Ne
    Jak? ________________________________________

A - Žádá informace?        □ Ano □ Ne

M - Něco nesedí?           □ Ano □ Ne
    Co? _________________________________________
```

**Verdikt:** □ Legitimní □ Podvod
