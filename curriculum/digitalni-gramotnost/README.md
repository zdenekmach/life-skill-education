# Modul: Digitální gramotnost

## Přehled

Modul Digitální gramotnost připravuje děti na bezpečný a zodpovědný život v digitálním světě. Učí je chránit svou digitální identitu, rozumět AI technologiím a používat technologie kreativně a eticky.

## Cíle modulu

Po absolvování modulu žáci:
1. Dokáží chránit svou digitální identitu a soukromí
2. Rozumí základům kybernetické bezpečnosti
3. Chápou, jak funguje AI a její limity
4. Umí technologie používat kreativně a produktivně
5. Jednají jako zodpovědní digitální občané

## Klíčové koncepty

### SAFE Framework (Bezpečnost online)

```
S = SECURE (Zabezpeč)
    Silná hesla, 2FA, aktualizace

A = AWARE (Buď ve střehu)
    Phishing, podvody, manipulace

F = FOOTPRINT (Digitální stopa)
    Co zůstává online navždy

E = ETHICS (Etika)
    Digitální občanství, respekt
```

### Příprava
- Ověř, že máš přístup k demonstračním účtům

- Nastav prostředí pro práci s AI nástroji

### Citlivá témata
- Kyberšikana může být osobní pro některé děti
- Respektuj různou úroveň technické gramotnosti
- Nezesměšňuj "základní" chyby (heslo123)
