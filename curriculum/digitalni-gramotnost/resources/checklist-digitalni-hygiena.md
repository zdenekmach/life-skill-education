# Checklist: Digitální hygiena

## Co je digitální hygiena?

Stejně jako si myješ ruce, abys předešel/a nemocem, digitální hygiena jsou návyky, které tě chrání online.

## Okamžité akce

### Když instaluji novou aplikaci

```
□ Je to oficiální aplikace? (ověřený vývojář)
□ Jaká oprávnění žádá? (dávají smysl?)
□ Jaké má recenze?
□ Potřebuji ji opravdu?
```

### Když dostanu podezřelou zprávu

```
□ STOP - neklikám hned
□ Kontroluji odesílatele
□ Hledám varovné znaky (tlak, chyby, divné odkazy)
□ Ověřuji přes oficiální kanály
□ Pokud podvod → mažu, blokuji, nahlašuji
```

### Když se registruji na nový web

```
□ Potřebuji tam být?
□ Jaké údaje požaduje? (minimum nutných)
□ Použiji silné, unikátní heslo
□ Zapnu 2FA, pokud je dostupné
□ Přečtu si základy o soukromí
```
