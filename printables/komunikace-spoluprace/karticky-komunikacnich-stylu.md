# Kartičky komunikačních stylů a situací

## Jak používat

- Vytiskni, rozstříhej a (volitelně) zalaminuj
- Použij pro cvičení rozpoznávání stylů
- Hrajte "Jaký je to styl?"
- Procvičujte přeformulování na asertivní

## Kartičky komunikačních stylů (6)

### Kartička 1: PASIVNÍ styl

**Přední strana:**
```
┌─────────────────────────────────┐
│                                 │
│          PASIVNÍ                │
│            🐢                   │
│                                 │
│     "Já nejsem důležitý/á"      │
│                                 │
└─────────────────────────────────┘
```

**Zadní strana:**
```
┌─────────────────────────────────┐
│                                 │
│  ZNAKY:                         │
│  • Ustupuji                     │
│  • Mlčím                        │
│  • Omlouvám se za všechno      │
│  • Vyhýbám se konfliktu        │
│                                 │
│  VÝSLEDEK:                      │
│  Frustrace, nevyslyšení         │
│                                 │
│  TYPICKÉ FRÁZE:                 │
│  "Je mi to jedno"               │
│  "Jak chceš"                    │
│  "Promiň, že..."                │
│                                 │
└─────────────────────────────────┘
```

## Hry s kartičkami

### Hra 1: Roztřiď styly
Promíchej kartičky reakcí, děti třídí do 3 hromádek (pasivní, asertivní, agresivní).

### Hra 2: Přeformuluj
Vytáhni situaci, každý formuluje JÁ-výrok. Hlasujte o nejlepším.

### Hra 3: WIN-WIN challenge
Dvojice vytáhne konflikt a má 3 minuty najít WIN-WIN řešení.

### Hra 4: Role-play
Jeden hraje situaci pasivně/agresivně, druhý ukazuje asertivní alternativu.
