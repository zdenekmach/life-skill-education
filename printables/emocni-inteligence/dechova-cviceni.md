# Dechová cvičení pro emoční regulaci

> Praktický průvodce 5 dechovými technikami

---

## Proč dýchání funguje?

Když jsme ve stresu, dýcháme rychle a povrchně. To říká mozku: "NEBEZPEČÍ!"

Když **vědomě zpomalíme dech**, říkáme mozku: "Je klid, jsme v bezpečí."

**Výdech > Nádech** = Aktivuje "odpočinkový" nervový systém

---

## Technika 1: Box Breathing (Dýchání do čtverce)

### Pro co je dobrá
- Obecné uklidnění
- Soustředění
- Před testem, vystoupením

### Jak na to

```
         NÁDECH (4 sekundy)
         ──────────────────→
         |                 |
         |                 |
ZADRŽET  |                 | ZADRŽET
(4 sek)  |                 | (4 sek)
         |                 |
         |                 |
         ←──────────────────
         VÝDECH (4 sekundy)
```

### Instrukce

1. Sedni si pohodlně, nohy na zemi
2. Zavři oči (nebo se dívej na jedno místo)
3. **NÁDECH** nosem: 1... 2... 3... 4...
4. **ZADRŽET** (jemně): 1... 2... 3... 4...
5. **VÝDECH** ústy: 1... 2... 3... 4...
6. **ZADRŽET** (prázdné plíce): 1... 2... 3... 4...
7. Opakuj 4-6 cyklů

### Tip
Představ si, že kreslíš čtverec dechem - každá strana = jeden krok.

## Technika 2: 5-5-7 Dýchání (Uklidňující)

### Pro co je dobrá
- Úzkost, nervozita
- Před spaním
- Po konfliktu

### Proč funguje
Delší výdech (7s) než nádech (5s) aktivuje parasympatický nervový systém = "brzdový pedál" těla.

### Instrukce

1. **NÁDECH** nosem: 1... 2... 3... 4... 5...
2. **ZADRŽET** jemně: 1... 2... 3... 4... 5...
3. **VÝDECH** ústy, pomalu: 1... 2... 3... 4... 5... 6... 7...
4. Opakuj 5-8 cyklů

### Tip
Při výdechu si představ, že pouštíš napětí ven jako páru z hrnce.

---

## Technika 3: Balloon Breathing (Balonkové dýchání)

### Pro co je dobrá
- Pro mladší děti nebo začátečníky
- Zábavná forma
- Břišní dýchání

### Instrukce

1. Polož ruku na břicho
2. Představ si, že v břiše máš balónek
3. **NÁDECH**: Nafukuješ balónek - břicho se zvedá
4. **VÝDECH**: Pomalu vypouštíš vzduch - břicho klesá
5. Opakuj pomalu, 6-8 cyklů

### Tip
Můžeš při tom dělat zvuk "ssssss" jako unikající vzduch z balónku.

---

## Technika 4: Grounding Breath (Uzemnění)

### Pro co je dobrá
- Panika, disociace ("jsem mimo")
- Pocit odpojení od těla
- Po noční můře

### Instrukce

1. Stůj nebo seď s nohama na zemi
2. **NÁDECH**: Představ si, že vdechuješ energii ZE ZEMĚ nahoru
3. **VÝDECH**: Představ si, že vydechuješ napětí ZPĚTdo země
4. Při nádechu si všímej nohou na zemi
5. Opakuj, dokud se necítíš "tady a teď"

### Rozšíření: 5-4-3-2-1

Kombinuj s grounding technikou:
1. Po 3 cyklech dýchání...
2. Pojmenuj 5 věcí, které VIDÍŠ
3. 4 věci, které SLYŠÍŠ
4. 3 věci, kterých se DOTÝKÁŠ
5. 2 vůně
6. 1 chuť

## Technika 5: Energizující dýchání

### Pro co je dobrá
- Únava, ospalost
- Potřeba energie
- Ráno pro probuzení

### Instrukce (Kapalabhati - "zářivá lebka")

⚠️ Pouze pro starší děti, krátce, s dospělým!

1. Sedni si vzpřímeně
2. Krátký, silný VÝDECH nosem (jako bys sfoukl svíčku)
3. Nádech přijde automaticky, pasivně
4. Opakuj 10-20 rychlých výdechů
5. Pak normálně dýchej

### Jednodušší varianta: Lví dech

1. **NÁDECH** nosem
2. **VÝDECH** ústy s jazykem ven a zvukem "HAAA"
3. Opakuj 3x

---

## Kdy kterou techniku použít

| Stav | Technika |
|------|----------|
| Nervozita před akcí | Box Breathing |
| Úzkost, strach | 5-5-7 |
| Hněv (před explozí) | 5-5-7 nebo Box |
| Panika | Grounding + 5-4-3-2-1 |
| Únava | Energizující nebo Lví dech |
| Před spaním | 5-5-7 |
| Pro soustředění | Box Breathing |

## Tipy pro úspěch

1. **Trénuj v klidu** - Nejdřív cvič, když jsi v pohodě. Pak to půjde i ve stresu.

2. **Pravidelnost** - 2 minuty denně je lepší než 20 minut jednou za týden.

3. **Dýchej do břicha** - Břicho se má zvedat, ne ramena.

4. **Nepřehánět** - Pokud se ti točí hlava, vrať se k normálnímu dýchání.

5. **Kombinuj** - Dýchání + vizualizace = silnější efekt.
